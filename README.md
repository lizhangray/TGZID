# TGZID

## [ICME 2026] 《Text-Guided Zero-Shot Image Dehazing》

------

> **Abstract:** *Zero-shot dehazing directly utilizes the input image without external datasets, offering a promising solution for real-world applications. The main challenge is extracting sufficient information from a hazy image, as haze significantly covers visual details. To address this issue, we propose a text-guided zero-shot image dehazing (TGZID) network, which is trained using only one hazy image and guided by haze-aware textual descriptions through self-supervised learning. A vision-language contrastive learning module aligns image and text representations in a multimodal embedding space. Positive (haze-free) and negative (hazy) descriptions are selected via a random sample screening algorithm. To better leverage multimodal semantic information, we further design a text-guided multiscale feature enhancement module, which incorporates edge attention blocks to refine feature fusion. To our knowledge, our approach is the first to implement vision-language guidance for zero-shot image dehazing. Extensive experiments demonstrate that our approach outperforms several state-of-the-art methods. The code is available at https://github.com/lizhangray/TGZID.*
